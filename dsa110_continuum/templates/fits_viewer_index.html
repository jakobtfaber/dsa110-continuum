FITS Viewers

Web Viewer

Interactive FITS file browser with support for JS9, CARTA, and Aladin viewers.

→ Open Web Viewer

Supported Viewers

  • JS9 (default) - Interactive FITS viewer
  • CARTA - Cube Analysis and Rendering Tool for Astronomy
  • Aladin Lite - Sky visualization and exploration

API Documentation

The FITS viewer is accessible via REST API endpoints at /api/v1/fits:

  • /api/v1/fits/viewer-urls - Get viewer URLs for a FITS file
  • /api/v1/fits/metadata - Extract FITS header metadata
  • /api/v1/fits/preview-html - Get HTML preview with viewer buttons
  • /api/v1/fits/validate - Validate a FITS file
  • /api/v1/fits/config - Get viewer configuration
  • /api/v1/fits/css - Get CSS styles
