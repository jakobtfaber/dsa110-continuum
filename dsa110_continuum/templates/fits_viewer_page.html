API Documentation

This viewer uses the DSA-110 FITS API to provide interactive access to FITS files. Available endpoints:

  • {escaped_api}/viewer-urls - Get viewer URLs for a FITS file
  • {escaped_api}/metadata - Extract metadata from FITS header
  • {escaped_api}/preview-html - Get HTML preview with viewer buttons
  • {escaped_api}/validate - Validate a FITS file
  • {escaped_api}/config - Get viewer configuration
  • {escaped_api}/css - Get CSS styles for buttons

Base URL: {escaped_base_url}
